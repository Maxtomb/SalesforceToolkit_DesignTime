软件会定时运行发送http请求到指定的url上去 测试暂时不需要加密

Json例子:
{
"TestGreeting__Cloth__c":[
{"Name":"jsondata1","TestGreeting__cloth_style__c":"jsonstyle1","TestGreeting__cloth_color__c":"jsoncolor1"},
{"Name":"jsondata2","TestGreeting__cloth_style__c":"jsonstyle2","TestGreeting__cloth_color__c":"jsoncolor2"},
{"Name":"jsondata3","TestGreeting__cloth_style__c":"jsonstyle3","TestGreeting__cloth_color__c":"jsoncolor3"},
{"Name":"jsondata4","TestGreeting__cloth_style__c":"jsonstyle4","TestGreeting__cloth_color__c":"jsoncolor4"},
{"Name":"jsondata5","TestGreeting__cloth_style__c":"jsonstyle5","TestGreeting__cloth_color__c":"jsoncolor5"}],
"TestGreeting__SalesManName__c":[
{"Name":"jsondata1","TestGreeting__sales_phone__c":"12312312321"},
{"Name":"jsondata2","TestGreeting__sales_phone__c":"12332442242"},]
}

TestGreeting__Cloth__c 为Salesforce的表(sobject)的名字
Cloth 为数据库表名
TestGreeting__cloth_style__c 为Salesforce的表中的字段名
cloth_style 为数据库表字段名
